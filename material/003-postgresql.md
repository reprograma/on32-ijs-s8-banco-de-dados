# PostgreSQL

O **`PostgreSQL`** é um sistema de gerenciamento de banco de dados relacional de código aberto e altamente poderoso. Ele é conhecido por sua confiabilidade, robustez e recursos avançados. Ele surgiu a partir do projeto POSTGRES na Universidade da Califórnia, Berkeley, e tem sido desenvolvido ativamente desde então pela comunidade open-source.

## 👉 Principais Características do PostgreSQL

- **Suporte total a SQL:** Oferece uma implementação completa do padrão SQL com extensões.
- **Confiabilidade e estabilidade:** É reconhecido por sua estabilidade, robustez e capacidade de manipular grandes volumes de dados.
- **Extensibilidade:** Permite a adição de novos tipos de dados, funções e linguagens de programação.
- **Recursos avançados:** Suporta transações ACID, replicação, particionamento, e outras funcionalidades avançadas.
- **Compatibilidade com padrões:** Adere aos padrões ANSI SQL e SQL99, além de suportar recursos adicionais.

## 👉 Conceitos Fundamentais do PostgreSQL

- **Bancos de Dados e Esquemas:** Explicação sobre como o PostgreSQL organiza os dados em bancos de dados e esquemas.
- **Tabelas e Tipos de Dados:** Introdução às tabelas, colunas e tipos de dados suportados pelo PostgreSQL (inteiros, strings, datas, etc.).
- **Consultas SQL Básicas:** Exemplos de consultas SQL simples para recuperar dados de uma tabela.
- **Chaves Primárias e Estrangeiras:** Explicação sobre como definir e usar chaves primárias e estrangeiras para garantir a integridade dos dados.
- **Índices:** Conceito de índices e sua importância para otimizar consultas.

## 👉 Comandos iniciais sql

- **CREATE DATABASE**: Criar um novo banco de dados.
- **CREATE TABLE**: Criar uma nova tabela no banco de dados.
- **ALTER TABLE**: Adicionar, modificar ou excluir colunas de uma tabela existente.
- **DROP TABLE**: Excluir uma tabela do banco de dados.
- **SELECT**: Selecionar dados de uma ou mais tabelas.
- **INSERT INTO**: Adicionar novas linhas (registros) a uma tabela.
- **UPDATE**: Modificar valores de colunas em registros existentes em uma tabela.
- **DELETE FROM**: Excluir linhas (registros) de uma tabela.
- **CREATE INDEX**: Criar um novo índice em uma tabela.
- **DROP INDEX**: Excluir um índice existente de uma tabela.
- **WHERE**: Filtrar registros de uma tabela com base em uma condição.
- **JOIN**: Combinar dados de duas ou mais tabelas com base em uma condição de associação.
- **GROUP BY**: Agrupar linhas que possuem o mesmo valor em uma ou mais colunas e aplicar funções de agregação, como COUNT, SUM, AVG, etc.
- **ORDER BY**: Ordenar os resultados de uma consulta com base nos valores de uma ou mais colunas, em ordem ascendente ou descendente.
- **LIMIT**: Limitar o número de linhas retornadas por uma consulta.
